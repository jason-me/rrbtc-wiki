Entities Overview

Profiles of the projects, organizations, and software in the Bitcoin ecosystem. Each entry covers what the entity does, how it fits into the broader landscape, and where to learn more.

📄️Alby

Browser extension for Bitcoin Lightning payments and Nostr signing. Provides a non-custodial wallet with seamless web integration via the WebLN standard.

📄️Azteco

Azteco is a bitcoin voucher system for low-KYC onboarding. Buy a printed or digital voucher with cash or card, then redeem to any bitcoin address with no account registration.

📄️Base58

A Bitcoin protocol education organization offering intensive, in-person bootcamps focused on Bitcoin Core, transactions, and protocol fundamentals. Named after the Base58Check encoding scheme used in Bitcoin addresses.

📄️Bisq

Bisq is a decentralized, peer-to-peer exchange for buying and selling bitcoin with national currencies or other cryptocurrencies. No account creation, no KYC, no central server — trades are coordinated over Tor and settled via 2-of-2 multisig escrow with a third-party mediator.

📄️BitAxe

BitAxe is an open-source, low-power Bitcoin ASIC miner designed for home mining. It bridges the gap between industrial-scale equipment and hobbyist participation in network security.

📄️Bitcoin Core

Bitcoin Core is the reference implementation of the Bitcoin protocol. Originally released by Satoshi Nakamoto in 2009 as "Bitcoin," it remains the most widely used, reviewed, and trusted node software.

📄️Bitcoin Knots

Bitcoin Knots is an enhanced full-node implementation of Bitcoin, maintained by Luke Dashjr. It builds on Bitcoin Core with additional privacy, decentralization, and mining-related features.

📄️Blockstream Green

Blockstream Green is a mobile and desktop Bitcoin wallet with optional 2FA multisig and hardware wallet integration.

📄️BRAIINS Hashpower

On-demand SHA-256 hashrate marketplace launched by Braiins in January 2026 (public beta). Miners buy and sell hashrate through a live order book without long-term contracts or hardware ownership.

📄️Breez

Mobile Lightning wallet for iOS and Android. Built on Flutter. Non-custodial, Lightning-native, and includes a built-in marketplace plus point-of-sale (POS) support.

📄️CARAVAN

Open-source multisig coordinator by Unchained Capital. Web-based tool for building and managing multisig transactions without installing software.

📄️CASA

Fee-based collaborative custody service providing managed multisig vaults and personal security consulting.

📄️Cashu

Cashu is a free and open-source Chaumian ecash system built for Bitcoin. It lets users hold and transfer ecash tokens offline, settling against the Lightning Network when needed. Nobody needs to know who you are, how much you hold, or with whom you transact.

📄️c-lightning

Core-Lightning (CLN, often called c-lightning) is a lightweight, modular Lightning Network implementation. It is notable for early support of Bolt 12, which enables static Lightning offers for automated recurring payments.

📄️Coldcard

ColdCard is a hardware wallet for advanced users prioritizing security verification, air-gapped signing, and open-source principles. Manufactured by Coinkite, widely used in multisig setups.

📄️DATUM

DATUM (Decentralized Template Auction Machine / UTXO Manager) is a protocol that lets miners build custom block templates directly on their own node and submit them to a mining pool—without the pool dictating what transactions go into the block.

📄️Electrum

Electrum is one of the oldest Bitcoin wallets, using SPV to verify transactions without downloading the full blockchain. Its server implementation (ElectrumX, Electrs, Fulcrum) is widely bundled with node packages.

📄️Fold

Fold is a bitcoin rewards company offering a debit card and app for earning sats-back on everyday purchases.

📄️HashGuard

DIY Python auto-optimizer for the Braiins Hashpower marketplace. Monitors the order book, detects stalled bids, and automatically adjusts pricing to maintain hashrate flow while minimizing overpayment.

📄️Hashrate Autopilot

Official automation tool from Braiins for managing hashrate bids on the Braiins Hashpower marketplace. Rules-based engine that buys hashrate when price crosses thresholds and auto-rebalances between pools.

📄️HodlHodl

HodlHodl is a non-custodial, peer-to-peer bitcoin exchange platform. Unlike Bisq, it is web-hosted rather than fully decentralized, but it remains non-custodial through 2-of-3 multisig escrow.

📄️HWI

Command-line tool and Python library for connecting hardware wallets to Bitcoin Core. Enables direct signing with devices like Trezor, Ledger, and Coldcard without trusting intermediary software.

📄️JoinMarket

Decentralized CoinJoin marketplace. Uses a maker-taker model where liquidity providers (makers) earn fees for providing inputs to CoinJoin transactions, while privacy seekers (takers) pay for the service.

📄️Kaboom Racks

ASIC mining hardware marketplace. Sells refurbished and used miners, often at prices accessible to home miners.

📄️Ledger

Ledger is a hardware wallet manufacturer using secure element (SE) chips to protect private keys. One of the most widely sold hardware wallets globally.

📄️LNbits

An open-source Lightning accounting system that sits on top of a Lightning node. Provides a web interface, wallet system, and extensions for building Lightning-powered applications.

📄️Muun

Self-custodial Bitcoin wallet with a hybrid Lightning/on-chain model. Uses submarine swaps to enable Lightning payments without the user managing channels directly.

📄️MyNode

MyNode is a plug-and-play Bitcoin node OS similar to Umbrel, bundling Bitcoin Core, LND, Electrum, BTCPay, JoinMarket, Whirlpool, and more.

📄️Nunchuk

Mobile-first Bitcoin wallet focused on multisig and collaborative custody. Designed for shared vaults and inheritance planning.

📄️Ocean.xyz

Ocean.xyz is a Bitcoin mining pool that emphasizes transparency, non-custodial payouts, and miner sovereignty through the DATUM protocol.

📄️OpenNode

Bitcoin payment processor for businesses. Settles payments via Lightning and on-chain with an API, e-commerce plugins, and hosted payment pages.

📄️Oshi App

Bitcoin rewards and marketing platform for local businesses. Replaces traditional loyalty stars, punch cards, and points with bitcoin-backed incentives.

📄️Payjoin Dev Kit

Open-source SDK for building Payjoin (BIP 78) functionality into Bitcoin wallets and applications. Maintained by Dan Gould.

📄️Phoenix

Self-custodial Lightning wallet by ACINQ. Designed to abstract channel management from the user while keeping keys on-device.

📄️PlebDevs

A Bitcoin developer education initiative founded by Austin (@bitcoinplebdev) that teaches plebs to become technical builders on Bitcoin, Lightning, and Nostr. Operates out of PlebLab in Austin, TX.

📄️PlebLab

PlebLab is an Austin-based Bitcoin education initiative home to PlebDevs, a program that teaches Lightning and Bitcoin app development to beginners.

📄️RaspiBlitz

RaspiBlitz is a DIY Bitcoin Lightning node project built for education. Unlike commercial plug-and-play nodes, it emphasizes learning the underlying tools.

📄️RoboSats

RoboSats is a peer-to-peer exchange for buying and selling bitcoin using Lightning Network payments. It operates entirely over Tor and uses temporary robot identities, meaning users do not accumulate persistent trade history or reputation.

📄️RoninDojo

Privacy-focused Bitcoin node operating system. On-chain only, with integrations for Whirlpool (CoinJoin) and Bisq.

📄️Samourai Wallet

Privacy-focused Bitcoin wallet for Android, built around on-chain transaction privacy. The project was the origin of several privacy tools later adopted across the ecosystem.

📄️SeedSigner

Open-source, DIY Bitcoin signing device built from commodity hardware. Designed for air-gapped seed generation, transaction signing, and multisig coordination without trusting proprietary hardware.

📄️Simple Proof

Bitcoin-based document timestamping service that anchors cryptographic hashes of digital records to the Bitcoin blockchain, creating tamper-proof proof of existence and authenticity.

📄️Sparrow Wallet

Sparrow Wallet is a desktop Bitcoin wallet focused on privacy, security, and flexibility. It supports single-sig and multisig setups and is designed for users who want full control over their coins and connections.

📄️Specter Desktop

Open-source DIY multisig coordinator for Bitcoin. Runs as a desktop application with optional hardware wallet integration.

📄️StartOS

StartOS (formerly Start9OS) is a Debian-based operating system designed specifically for self-hosting Bitcoin and Lightning infrastructure. It packages complex services into one-click installs with automatic Tor and LAN access.

📄️Stratum V2

Stratum V2 (Sv2) is a next-generation mining protocol that addresses the centralization risks inherent in the original Stratum protocol by giving miners control over block template construction.

📄️Strike

Consumer-friendly Bitcoin and Lightning payment application. Allows users to buy, sell, and send bitcoin with a focus on simplicity and low fees.

📄️Tapsigner

NFC-enabled card from Coinkite storing a Bitcoin private key. Sign transactions by tapping the card to your phone.

📄️Trezor

Trezor is a hardware wallet developed by SatoshiLabs. The first commercial hardware wallet (2014), known for fully open-source firmware.

📄️Umbrel

Umbrel is a plug-and-play Bitcoin node OS designed for Raspberry Pi and x86 mini PCs. It bundles Bitcoin Core, Lightning implementations, and a growing app store.

📄️Unchained Capital

Collaborative custody provider and financial services company focused on bitcoin-native products.

📄️Wasabi Wallet

Privacy-focused Bitcoin desktop wallet with built-in CoinJoin coordination by default. Uses the WabiSabi credential protocol for CoinJoin transactions.

📄️Whirlpool

Whirlpool is a zero-link CoinJoin implementation developed by Samourai Wallet. It breaks deterministic links between inputs and outputs, improving on-chain privacy.

📄️Zeus

Mobile Lightning wallet and node management app. Connects to your own Lightning node (LND, Core Lightning, or Eclair) remotely, giving you full control without running a node on your phone.
