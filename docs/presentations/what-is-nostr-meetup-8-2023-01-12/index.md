# What Is NOSTR? Meetup #8 Summary
**Date:** January 12, 2023

## NOSTR
- Notes and Other Stuff Transmitted by Relays
- Decentralized network based on cryptographic keypairs
- Not peer-to-peer: users interact via relays
- Every user identified by Public Key; every post signed and validated
- NIP-01 lays out core protocol

## How It Works
1. Users run a client (app); anyone can run a relay (server)
2. Clients publish signed data to relays of their choice
3. To follow someone, client queries relays for posts by public key
4. On startup, client pulls data chronologically from known relays

## Key Management
- Alby: Chrome/Firefox extension for signing and wallet connection
- Nos2x: dedicated signing extension

## Advanced Concepts
- nostrnode-cli: send encrypted RPC commands to Bitcoin Core, Core Lightning, Join Market via Nostr events
- Nostr 2.0: Layer 2 off-chain data storage, incentivizing relays to become full nodes
