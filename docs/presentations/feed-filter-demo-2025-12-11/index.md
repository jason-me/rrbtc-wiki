## Outline & Notes

# Feed Filter Demo – 11 Dec 2025
**Speaker:** Ted Thayer (Pleb Lab)
**Date:** December 11, 2025

## Executive Summary
Feed Filter is a user-controlled AI filter for social media (initially X/Twitter). Restores "mental sovereignty" by letting users algorithmically filter feeds via browser extension and mobile app.

## Problem
- 1 in 5 people estranged from family due to social media disagreements
- Facebook algorithm weights rage-bait 5X higher to maximize user time
- Core question: "what if we could control which posts we see?"

## Technical Architecture
1. User sets filter rules
2. Network intercepted (X/Twitter)
3. Post data processed by nearest server (2x VLM local models, classified with reason)
4. Feed DOM altered as user visits site

## Competitive Positioning
| Feature | Native | Keyword | Feed Filter |
| Works across sites | ✗ | ✓ | ✓ |
| High user control | ✗ | ✗ | ✓ |
| Set & Forget | ✗ | ✓ | ✓ |
| Nuanced contextual understanding | ✓ | ✗ | ✓ |

## Roadmap
- Brand established, PoC built, infrastructure built
- Alpha/Beta testing on X/Twitter
- Public Chrome Webstore launch
- Add new platforms, marketing, features

## Strategic Focus
- Minimize reliance on APIs, VCs, regulatory chokepoints
- Browser and phone as control points
