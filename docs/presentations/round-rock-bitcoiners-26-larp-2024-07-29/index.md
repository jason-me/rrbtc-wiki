# Round Rock Bitcoiners #26 – LARP Presentation

**Date:** 2024-07-29  
**Presenter:** RRBTC  
**Topics:** larp, lightning, security, cosigners, lnbits

---

## Outline & Notes

## Summary
A role-playing exercise demonstrating a Lightning payment flow, followed by a discussion of Lightning security tools and cosigner options.

### LARP Scenario
The deck sets up a live-action roleplay for making a Lightning payment, though the specific transcribed dialogue was not captured in extraction.

### Lightning Security Discussion
- Lightning node security considerations
- Cosigner options for Lightning wallets
- Trade-offs between hot wallets and cold storage on Lightning

### Tools Referenced
- **LNbits** – Open-source Lightning accounting system with extensions
- Lightning cosigner services
- Multi-signature approaches for Lightning channels

### Context
This was Meetup #26 in the series. The presentation title includes "LARP" suggesting an interactive, participant-driven demonstration of Lightning payment flows rather than a traditional lecture format.
