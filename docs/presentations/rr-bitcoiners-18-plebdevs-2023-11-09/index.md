# RR Bitcoiners #18 – PlebDevs Takeover

**Date:** 2023-11-09  
**Presenter:** Austin (@bitcoinplebdev)  
**Topics:** plebdevs, lightning, development, pleblab, education

---

## Outline & Notes

## Summary
Austin from PlebLab presents the "PlebDev" concept and demonstrates how to build a simple Lightning wallet using Polar + Replit.

### What is a PlebDev?
> "A plebdev is a Bitcoin pleb turned technical"

### Evolution Stages
1. **The Pleb** – Learns Bitcoin fundamentals, first rabbit hole
2. **The Bitcoin Pleb** – Runs a node, writes first code
3. **The PlebDev** – Can write functional code, understands Bitcoin from first principles

### Core PlebDev Principles
- Targets all learning stages, from first-time coders to 2-year devs
- Focus: app development (protocol development also welcome)
- "Plebdevs embrace the label of n00b, being a noob is 1000x better than being on the sidelines"
- "Bitcoin/Lightning/Nostr needs more Plebdevs!"
- "Not everyone wants/needs to become a developer. But you can at least become a little dangerous!"

### Presenter Background (Austin)
- Mostly self-taught + fullstack bootcamp + Base58 Transactions bootcamp
- Active in Austin Bitcoin/Lightning dev scene
- Co-created PlebDevs Lightning course, runs public PlebDevs community
- Career: bought first Bitcoin → Antminer S9 (broke) → shitcoin trading (lost money) → learned to code → moved to Austin → first Bitcoin job at Fold → founded PlebDevs

### PlebDevs Offerings
- First PlebDevs Lightning development course
- Regular in-person and virtual workshops
- Public Discord community

### Case Studies
1. **PlebPoet** (plebpoet.com) – Writer/designer building her own website
2. **BTCRetriever** – Coding bootcamp grad leveling up Bitcoin-specific skills
3. **Bitdern** – Writer aiming to land a job in Bitcoin

### Tools Demonstrated

#### Polar (Local Lightning Simulator)
- Quick setup of local Lightning networks
- Manage multiple nodes (LND, Core Lightning, Eclair, Bitcoin Core)
- Create channels, invoices, payments
- Testing across node implementation versions
- Setup: Docker Desktop → Polar → Create Network → Start

#### Lightning Wallet Build Stack
- Replit (free web-based IDE)
- WebLN (API for web apps to interact with Lightning)
- Alby (Lightning wallet + browser extension)
- Demo code in vanilla JS and React
- Live demo: replit.com/@bitcoinplebdev/rr-plebdevs
