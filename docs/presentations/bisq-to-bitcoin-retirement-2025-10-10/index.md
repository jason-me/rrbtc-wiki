# From First Sats on Bisq to Bitcoin Retirement: A HODLer's Journey
**Author:** RRBTC

## Overview
Connects two ideas: buying bitcoin privately with Bisq, and holding it long term.

## Part 1: Acquiring Bitcoin Privately with Bisq
- Peer-to-peer, open-source, no-KYC Bitcoin exchange
- Runs over Tor for privacy
- Non-custodial — users always hold their own keys

### Bisq 1 vs. Bisq 2 (Easy)
| Feature | Bisq 1 | Bisq 2 |
| Model | Multisig escrow + deposit | Reputation-based (no deposit) |
| Trade size | Larger / advanced | Smaller / onboarding |
| Security | Escrow + penalty | Reputation + trade limits |

### GPG Verification
```bash
gpg --verify Bisq-2.X.X.dmg.asc Bisq-2.X.X.dmg
```

## Part 2: Bitcoin Retirement Strategy
- Shift from acquisition to preservation
- Self-custody and long-term security
- Daily/monthly DCA, lump sum, Roth IRA conversion

### Custody and Inheritance
- Start simple with bitcoin-only wallets
- Secure backups: hardware wallet + steel/paper copy
- Create inheritance plan (key shares or trusted provider)
- Avoid cloud storage or third-party dependence
- Revisit setup yearly, patch firmware

### Multi-Sig Tools
- Unchained Capital — collaborative custody (2-of-3)
- Specter Desktop — DIY multi-sig
- Nunchuk — mobile shared vaults
- Caravan — open-source coordinator

### Tools & Calculators
- Unchained Retirement Calculator
- Bitcoin Magazine CAGR Calculator
- Bitbo CAGR Charts
- Bitcoiner Academy Calculator
- Saylor's Bitcoin Model on GitHub
