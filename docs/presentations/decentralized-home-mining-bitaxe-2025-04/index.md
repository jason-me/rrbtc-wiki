# Decentralized Home Mining w/ Bitaxe
**Presenter:** RRBTC
**Date:** April 2025

## Evolution & Current State
Mining transitioned from CPUs to GPUs to FPGAs to ASICs to Industrial Facilities.
Bitaxe lowers barriers, supporting open-source innovation.
Network hashrate surpassed 1 Zetahash.

## Hardware Comparison
| Feature | Bitaxe 601 Gamma | Antminer S19j Pro+ |
| Hash Rate | 1.2 TH/s | 120 TH/s |
| Power Consumption | ~18 watts | 3,250 to 4,080 watts |
| Efficiency | 15 J/TH | 27 to 34 J/TH |
| Monthly Cost | ~$1.30 - $2.60 | ~$326.59 - $411.84 |

## Mining Strategies
### Pooled Mining
- PPS/FPPS: fixed payout per share
- PPLNS: based on recent shares
- TIDES (Ocean.xyz): transparent, predictable

### Solo Mining
- Full block reward, no pool fees
- 12 blocks solo mined in 2024; 2 by Bitaxes
- Use SoloChance.com for odds

## Setup: Bitaxe to Solo Pool
1. Power on, access IP shown on screen
2. Settings > Network > enter Wi-Fi
3. Stratum Host: public-pool.io, Port: 21496
4. User: [Your Bitcoin Address].[Worker Name]
5. Save & Restart

## Advanced Decentralized Stack
- StartOS: home server OS
- Bitcoin Knots: enhanced privacy/block template options
- C-Lightning: Bolt12 for Lightning payouts
- AxeOS: custom Bitaxe firmware
- DATUM (Ocean.xyz): custom work issuance

## DATUM Solo Mining Config
- Remove text from "Datum Pool Host"
- Disable "Failover To Lottery" if not wanted
- Disable "Pooled Mining Only"
