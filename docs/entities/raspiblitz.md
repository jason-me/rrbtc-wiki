# RaspiBlitz

RaspiBlitz is a DIY Bitcoin Lightning node project built for education. Unlike commercial plug-and-play nodes, it emphasizes learning the underlying tools.

## Properties
| Property | Detail |
|----------|--------|
| Hardware | Raspberry Pi 4/5 with SSD |
| Philosophy | DIY, transparent, educational |
| Setup | Script-based; some command-line required |
| License | MIT |

## Key Features
- LCD display for real-time status
- Bitcoin Core + LND + BTCPay + RTL
- Tor-only remote access
- Docs explain *why* each service runs

> Round Rock meetup note: "RaspiBlitz: DIY, fully open-source, Raspberry Pi"
