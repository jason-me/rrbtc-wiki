# Specter Desktop

Open-source DIY multisig coordinator for Bitcoin. Runs as a desktop application with optional hardware wallet integration.

## Features

- **Multisig coordination** — Build and manage M-of-N setups
- **Hardware wallet support** — Works with most major hardware wallets
- **Air-gapped option** — Can run on an offline machine
- **Full node integration** — Connects to Bitcoin Core or other backends
- **PSBT workflow** — Uses Partially Signed Bitcoin Transactions for air-gapped signing

## Use Cases

- Personal sovereign multisig vaults
- Family or small-group shared custody
- Inheritance planning with geographically distributed keys

## Trade-offs

- Requires technical setup and ongoing maintenance
- User is fully responsible for backup and recovery
- No recovery service if keys are lost
