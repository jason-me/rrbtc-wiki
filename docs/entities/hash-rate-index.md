# HashRate Index

Data and analytics platform for Bitcoin mining. Tracks miner revenue, profitability, power consumption by rig, and historical ASIC pricing.

## Key Data Sets
- **Miner Revenue**: Daily revenue per TH/s based on current difficulty and BTC price
- **Profitability by Rig**: Break-even electricity cost per ASIC model
- **Power Consumption**: Wattage and efficiency (J/TH) for current and legacy hardware
- **Historical ASIC Prices**: Track how miner hardware prices fluctuate with market cycles
- **Bitcoin Mining Profitability Calculator**: Model ROI with custom electricity rates and hardware costs

## Why It Matters for Home Miners
- Bear-market ASIC prices often collapse while revenue stays above electricity cost for efficient units
- Historical data reveals when S19J Pro+ or similar units become "hobby-viable"
- Power consumption tables help size residential electrical upgrades correctly

## Trade-offs
| Advantage | Limitation |
|-----------|------------|
| Comprehensive ASIC database | Data is aggregate, not real-time pool-specific |
| Historical trend analysis | Assumes average pool luck; solo miners see more variance |
| Free tier available | Advanced analytics require subscription |
