# MyNode

MyNode is a plug-and-play Bitcoin node OS similar to Umbrel, bundling Bitcoin Core, LND, Electrum, BTCPay, JoinMarket, Whirlpool, and more.

## Key Features
- **Web dashboard** — manage services and monitor sync status
- **One-click upgrades**
- **Tor by default**
- Available in free (community) and premium editions
