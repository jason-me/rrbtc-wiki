# RoninDojo

Privacy-focused Bitcoin node operating system. On-chain only, with integrations for Whirlpool (CoinJoin) and [Bisq](/entities/bisq).

## Features

- Full Bitcoin Core node with privacy defaults
- Integrated Whirlpool for CoinJoin liquidity
- Bisq integration for P2P trading
- Designed for use with privacy-focused wallets like Samourai
- Self-hosted: user maintains complete control over keys and transaction data
- Tanto: pre-built plug-and-play node hardware available

## Philosophy

- On-chain privacy first
- No Lightning by default (focus on base-layer privacy)
- Minimize metadata leakage to third-party servers
- Encourages running your own Dojo rather than relying on Samourai's infrastructure

## Comparison
