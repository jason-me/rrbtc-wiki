# IBEXPay

Merchant payment processor for accepting Bitcoin via Lightning and on-chain. Targets businesses from single-location shops to multi-location chains.

## Features
- Instant Lightning Network and on-chain payments
- No-code integration for over-the-counter sales
- Multi-location/multi-manager support
- API available for custom integrations

## Comparison to Alternatives
| Feature | IBEXPay | BTCPayServer | OpenNode |
|---------|---------|--------------|----------|
| Self-hosted option | No | Yes | No |
| Lightning + On-chain | Yes | Yes | Yes |
| Multi-location | Yes | Via plugins | Yes |
| Custody | Custodial | Self-custody | Custodial |

## Use Case
- Ideal for businesses wanting quick setup without running infrastructure
- Complements BTCPayServer for merchants preferring managed services
