# PlebLab

PlebLab is an Austin-based Bitcoin education initiative home to PlebDevs, a program that teaches Lightning and Bitcoin app development to beginners.

## Key Programs
- **PlebDevs** — Lightning development course
- **Workshops** — In-person and virtual
- **Public Discord** — Peer support

## Origins
Founded by Austin, a self-taught developer who attended Base58, worked at Fold, and committed to growing the Bitcoin developer pipeline in Central Texas.
