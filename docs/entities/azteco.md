# Azteco

Azteco is a bitcoin voucher system for low-KYC onboarding. Buy a printed or digital voucher with cash or card, then redeem to any bitcoin address with no account registration.

## How It Works
1. Purchase voucher (cash at retail or online)
2. Receive redemption code or QR
3. Visit azte.co and enter code
4. Paste a bitcoin receiving address
5. Bitcoin sent directly to wallet

## Use Cases
- **Tipping** — easy to print and hand out
- **Onboarding** — recipient learns after receiving
- **Circular economy** — functions as primitive ecash locally
- **Remittances** — sender buys, recipient redeems globally
