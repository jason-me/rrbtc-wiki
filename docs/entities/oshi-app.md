# Oshi App

Bitcoin rewards and marketing platform for local businesses. Replaces traditional loyalty stars, punch cards, and points with bitcoin-backed incentives.

## How Businesses Use It
- Customers earn bitcoin instead of coffee points
- Built-in local map showing participating merchants
- Marketing tool to attract the Bitcoin community
- Some locations listed offer gift card purchases as well

## Onboarding Value
- Lower barrier than full Lightning integration (no node needed)
- Immediate consumer incentive (earn sats, not points)
- Community effect: map draws Bitcoin users to listed businesses

## Trade-offs
| Advantage | Limitation |
|-----------|------------|
| Easy setup | Depends on consumer adoption of Bitcoin |
| Loyalty rewards in sats | Custodial/gift-card model for some merchants |
| Local discovery via map | Data may lag; verification encouraged |
