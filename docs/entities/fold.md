# Fold

Fold is a bitcoin rewards company offering a debit card and app for earning sats-back on everyday purchases.

## Properties
| Property | Detail |
|----------|--------|
| Product | Debit card + mobile app |
| Rewards | Sats-back on purchases |
| Onboarding | KYC required |
| Spin game | Gamified rewards wheel |
