# Casa

Fee-based collaborative custody service providing managed multisig vaults and personal security consulting.

## Model

- **2-of-3** or **3-of-5** multisig configurations
- You hold the majority of keys; Casa holds one recovery key
- Mobile app guides setup and key health monitoring
- Subscription-based service with tiered plans

## Services

- Personal vaults (2-of-3)
- Family vaults (3-of-5)
- Inheritance planning and key recovery
- Security consulting and health checks
- Mobile key on phone + hardware wallet keys

## Differentiation

- Mobile-first experience with guided setup
- Proactive security monitoring and alerts
- Inheritance protocol with legal documentation support
- No single point of failure — Casa cannot unilaterally move funds

## Trade-offs

- Subscription fees
- Requires trust in Casa's key recovery process
- Less privacy than fully self-hosted solutions
