# Strike

Consumer-friendly Bitcoin and Lightning payment application. Allows users to buy, sell, and send bitcoin with a focus on simplicity and low fees.

## Features

- **Bitcoin purchases** — Link bank account to buy bitcoin directly
- **Lightning payments** — Send and receive instant, low-cost payments
- **Cash out** — Convert bitcoin back to fiat and withdraw to bank
- **Global transfers** — Send money internationally via Lightning rails
- **Merchant tools** — Point-of-sale and payment request features

## Model

- Custodial by default — Strike holds bitcoin on behalf of users
- KYC required for fiat on/off ramps
- Non-custodial Lightning address option available
- Focus on user experience over sovereignty

## Trade-offs

- Custodial model means users do not hold private keys
- KYC requirements for fiat integration
- Convenience and speed vs. self-custody and privacy
- Best for onboarding and everyday payments, not long-term savings
