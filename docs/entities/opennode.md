# OpenNode

Bitcoin payment processor for businesses. Settles payments via Lightning and on-chain with an API, e-commerce plugins, and hosted payment pages.

## Product Suite
- **API**: Custom payment flows and integrations
- **E-commerce plugins**: Drop-in checkout for online stores
- **Hosted payment pages**: No-code addition to existing sites
- **Payouts**: Send bitcoin payments programmatically

## Positioning
- Lightning-first with on-chain fallback
- Developer-friendly with modern API
- Custodial model (funds held by OpenNode until withdrawal)

## Comparison to Alternatives
| Feature | OpenNode | IBEXPay | BTCPayServer |
|---------|----------|---------|--------------|
| API | Yes | Yes | Yes (REST) |
| Hosted pages | Yes | Yes | Via third-party |
| Custody | Custodial | Custodial | Self-custody |
| Lightning focus | Yes | Yes | Yes |
