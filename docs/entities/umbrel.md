# Umbrel

Umbrel is a plug-and-play Bitcoin node OS designed for Raspberry Pi and x86 mini PCs. It bundles Bitcoin Core, Lightning implementations, and a growing app store.

## Properties
| Property | Detail |
|----------|--------|
| Hardware | Raspberry Pi 4/5, x86 mini PCs |
| Ease | GUI-first setup |
| Apps | Lightning, Electrum server, BTCPay, mempool, etc. |

## Key Features
- **One-click app installation**
- **Tailscale VPN** — easy remote access
- **SSD required** — faster than SD card

> Round Rock meetup note: "Umbrel / MyNode / Nodl: plug and play, Lightning enabled"
