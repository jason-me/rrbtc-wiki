# Blockstream Green

Blockstream Green is a mobile and desktop Bitcoin wallet with optional 2FA multisig and hardware wallet integration.

## Properties
| Property | Detail |
|----------|--------|
| Type | Mobile (iOS, Android) and Desktop |
| Custody | Self-custody with optional 2-of-2 multisig |
| Hardware | Ledger via HWI; Blockstream Jade natively |
| Features | Liquid sidechain, Tor, customizable fees |

## Key Features
- **2FA multisig** — Blockstream holds one key as 2FA service
- **Blockstream Jade** — companion hardware wallet, air-gapped via QR/USB
- **Liquid Network** — confidential transactions on sidechain
