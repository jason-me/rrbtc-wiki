# Nunchuk

Mobile-first Bitcoin wallet focused on multisig and collaborative custody. Designed for shared vaults and inheritance planning.

## Features

- **Multisig vaults** — Create and manage M-of-N setups on mobile
- **Collaborative custody** — Share vaults with family or advisors
- **Inheritance planning** — Built-in tools for inheritance workflows
- **Air-gapped signing** — Support for hardware wallets and air-gapped signers
- **PSBT workflow** — Partially Signed Bitcoin Transaction handling

## Use Cases

- Family savings vaults
- Business treasury management
- Inheritance planning with time-locked releases
- Collaborative custody with trusted third parties

## Platform

- iOS and Android
- Mobile-first design
- Hardware wallet integration (Coldcard, Keystone, etc.)
