# BTCMap

Open-source project mapping Bitcoin-accepting merchants and locations worldwide. Community-driven data using OpenStreetMap as the base layer.

## Features

- **Interactive map** — Search and browse bitcoin-accepting businesses globally
- **Community editing** — Anyone can add or update merchant locations
- **OpenStreetMap integration** — Uses OSM data and tagging conventions
- **API available** — Query locations programmatically
- **Mobile apps** — iOS and Android apps for on-the-go discovery

## Tagging

Merchants are tagged with payment methods accepted:
- On-chain bitcoin
- Lightning Network
- Contactless (NFC)
- Other methods

## Use Cases

- Find local businesses accepting bitcoin
- Plan travel around bitcoin-friendly locations
- Verify merchant claims before visiting
- Discover circular economy opportunities
