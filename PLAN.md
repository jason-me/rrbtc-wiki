5. Generation script lives at `scripts/gen-slide-pages.js` (or .py) and is idempotent so future decks can be added by dropping a folder and re-running.
6. Existing outline pages (mining-like-the-big-players-2026-05-14.md etc.) remain as docs; cross-link outline <-> deck where both exist.
